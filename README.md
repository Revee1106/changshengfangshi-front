# changshengfangshi-front

《长生坊市》前端工程。

这是一个基于 Next.js 的现代修仙 Dashboard 前端，用于展示角色成长、修炼、心神采集、行囊和日志等低频长期养成玩法界面。当前版本以静态 mock data 为主，暂不接入真实后端。

## 当前版本

- 版本：0.0.2

## 技术栈

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- lucide-react

## 当前已完成内容

- 基础应用壳：桌面端左侧导航、移动端顶部资源条、移动端底部 Tab 导航。
- 总览页：四维概览、最近日志展示。
- 修炼页：修行方向选择、修炼周数输入、安全修炼和野外修炼地点展示。
- 心神页：辅修方向入口、采集地点选择与消耗/产物信息展示。
- 行囊页：物品分类筛选、丹药/灵草/灵矿/工具/杂物展示、使用和出售按钮状态。
- 角色页：基础信息、寿数命元、境界修为、四维属性、道心状态和成长统计。
- 突破页：突破风险、成功率、突破丹药选择和开始突破入口。
- 左侧角色状态卡：修为满足条件时显示“可突破”气泡入口。
- 地点卡片：修炼场所与心神采集地点统一为选中大卡、未选中小卡的响应式样式。
- UI 基础组件：Card、Button、Badge、Progress，以及资源卡、属性卡、日志列表等游戏组件。
- 静态数据：角色资料、资源、修炼地点、采集地点、行囊物品和日志 mock data。

## 本地开发

安装依赖：

```bash
npm install
```

启动开发服务：

```bash
npm run dev
```

常用检查：

```bash
npm run lint
npm run typecheck
npm run build
```
