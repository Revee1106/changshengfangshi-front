# AGENTS.md

## 项目背景

本项目是《长生坊市》，一款长期在线网页修仙养成游戏，玩法参考 Torn City 的低频长期成长结构，但主题是东方修仙。

前端目标是现代修仙 Dashboard，不是传统重美术手游界面。

## 技术栈

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- TanStack Query
- Zustand

## UI 风格

- 现代修仙
- 信息清晰
- 卡片式布局
- 高信息密度但不要拥挤
- 浅色优先
- 使用青绿、墨绿、玉色、琥珀色作为主要氛围色
- 不要使用厚重页游金色 UI
- 不要满屏复杂纹理
- 不要依赖大插画

## 页面布局

桌面端：
- 左侧导航
- 中间主内容区
- 右侧状态 / 日志栏

移动端：
- 顶部核心资源条
- 单列卡片
- 底部 Tab 导航

PC 端：
- UI 必须自适应不同尺寸显示器，桌面布局不得依赖固定宽度堆叠。
- 使用 `minmax(0, ...)`、响应式断点和可换行网格，避免模块溢出、错位或横向裁切。

## 核心资源命名

必须使用以下中文命名：

- 寿数
- 命元
- 心神
- 道心
- 修为
- 法力
- 神识
- 体魄
- 遁法
- 灵石

## 开发规则

- 优先实现静态假数据 UI，不要接真实后端。
- 不要引入新的大型 UI 框架。
- 优先复用现有组件。
- 页面文案使用中文。
- 所有常用操作按钮要显示消耗和预估收益。
- 风险行为需要明显提示，例如野外修炼、突破。
- 修改后请运行 lint / typecheck / build 中项目已有的检查命令。
